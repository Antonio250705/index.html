XPath: Navegación y Selección de Nodos

XPath es un lenguaje de expresiones utilizado para localizar y seleccionar nodos en un documento XML, tratándolo como una estructura jerárquica de árbol.

1. El Modelo de Árbol

XPath ve al XML como una familia de nodos. Contiene:

  • Nodo Raíz: El origen del documento (/).
  • Elementos: Las etiquetas del XML.
  • Atributos: Propiedades de las etiquetas (@).
  • Nodos de Texto: El contenido real.

2. Operadores de Selección

Sintaxis Descripción
/ Selección desde la raíz.
// Selección en cualquier nivel.
. Nodo actual.
.. Nodo padre.
@ Atributos.

3. Ejemplos de Expresiones XPath

Aplicando predicados y funciones para filtrar datos específicos:

                1. Seleccionar todos los coches: //coche
                2. El segundo coche de la lista: //coche[2]
                3. Coches con matrícula específica: //coche[@matricula="8280KWT"]
                4. Marca de coches cuyo combustible no sea el del último coche:
                //coche[combustible != //coche[last()]/combustible]/marca
