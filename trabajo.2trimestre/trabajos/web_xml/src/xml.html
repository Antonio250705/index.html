Resumen de XML

Es un lenguaje de marcado de propósito general definido por el W3C. El propósito principal del lenguaje es compartir datos a través de diferentes sistemas, como Internet. Es similar a HTML y no está predefinido, es decir,las etiquetas se definen.

Características de XML

XML es un lenguaje de marcado que permite estructurar datos de manera clara y legible. Sus principales características son:

  • Lenguaje de estructura libre. No está predefinido. Las etiquetas se definen.
  • Consta de dos partes, el prólogo y el cuerpo
  • Usa espacios de nombres para evitar ambigüedades
  • Su sintaxis es como la de HTML.
  • Debe cumplir las reglas w3c para estar bien formado
  • Utiliza esquemas en forma de árbol con una única raíz.

Estructura de XML

La estructura de un documento XML consta de un prólogo y un cuerpo.Vamos a ver sus dos partes:

PRÓLOGO
  • Declaración XML
  • Información del documento
    • Hojas de estilo XSLT
    • Hojas de estilo CSS
    • Validacion DTD (interna o externa)
CUERPO
  • Raíz. Árbol de contenido
    • Espacio de nombres
    • Validación de XSD
  • Elementos XML

Reglas sintácticas de XML

No es como HTML. Si no está bien formado dará error. Debes tener en cuenta las siguientes instrucciones:

Prólogo Debe estar definido con su versión y codificación correspondiente.

Jerarquía Estructura de árbol con un único nodo raíz obligatorio.

Emparejamiento Todas las etiquetas deben tener su apertura y cierre correspondiente.

Atributos Los valores de los atributos deben ir siempre entre comillas (simples o dobles).

Case Sensitive El lenguaje distingue estrictamente entre mayúsculas y minúsculas.

Nomenclatura Los nombres de etiquetas siguen reglas similares a las variables en programación.
